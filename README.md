# YCraft Tools

Professional After Effects extensions and scripts for motion designers.

## Products

- **STRUCT** — Layer management, comp cleanup, and project utilities
- **TextFlow Pro** — Text layer organization and styling
- **ExpressionKit** — Expression editor and debugger (Coming Soon)

## Development

No build step required. Open `index.html` in a browser.

For local development with live reload:
```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```
